Shari Follower Analyzer - 개인정보 보호 정책

1. 개요

Shari Follower Analyzer는 Threads.com에서 팔로워 분석 기능을 제공하는 Chrome 확장 프로그램입니다. 사용자의 개인정보 보호를 최우선으로 합니다.

2. 수집하는 정보

2.1 자동 수집 정보

  • 팔로워 목록 데이터: Threads.com에서 공개적으로 표시되는 팔로워의 사용자명과 프로필 정보
  • 로컬 저장 데이터: 분석 결과는 사용자 기기의 로컬 스토리지에만 저장

2.2 수집하지 않는 정보

  • 개인 식별 정보 (이메일, 전화번호, 주소 등)
  • 비공개 메시지나 DM
  • 패스워드나 인증 토큰
  • 사용자의 브라우징 히스토리

3. 정보 사용 목적

  • 팔로워 분석: 상호 팔로우 관계 분석 및 표시
  • 사용자 편의성: 팔로우하지 않는 사용자 식별 및 하이라이트
  • 로컬 데이터 저장: 분석 결과의 임시 저장 (기기 내에서만)

4. 데이터 보호

  • 외부 전송 없음: 수집된 데이터는 외부 서버로 전송되지 않습니다
  • 로컬 저장: 모든 데이터는 사용자 기기의 Chrome 로컬 스토리지에만 저장
  • 사용자 제어: 언제든지 확장 프로그램을 제거하여 저장된 데이터를 완전히 삭제할 수 있습니다

5. 데이터 보존

  • 데이터는 확장 프로그램이 설치된 동안에만 로컬에 보존됩니다
  • 확장 프로그램 제거 시 모든 데이터가 자동으로 삭제됩니다
  • 사용자가 "데이터 리셋" 기능을 통해 언제든 삭제 가능합니다

6. 사용자 권리

  • 접근권: 저장된 데이터 확인 가능
  • 삭제권: 언제든지 데이터 삭제 가능
  • 제어권: 확장 프로그램 비활성화/제거 자유

7. 보안

  • HTTPS 통신만 사용
  • 로컬 스토리지 암호화 적용
  • 외부 API 호출 없음

8. 정책 변경

본 정책이 변경될 경우 확장 프로그램 업데이트를 통해 사용자에게 알림합니다.
